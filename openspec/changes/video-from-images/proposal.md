## Why

Users frequently create slideshow videos from their photo collections for sharing or archiving. Building this feature directly into the photo manager eliminates the need for external video editing software for the common case of a simple image sequence with background music.

## What Changes

- A new "Create Video" wizard lets users select an ordered list of images, set a per-slide duration, and optionally choose a background music file
- The backend invokes FFmpeg via `ProcessBuilder` to combine the images and audio into an MP4 file
- Progress is streamed via SSE (same pattern as catalog/sync/convert)
- The output is saved to a user-chosen folder and auto-cataloged as a video asset
- Requires `video-file-support` (#21) for FFmpeg presence in the container

## Capabilities

### New Capabilities

- `video-from-images`: Users can create an MP4 slideshow video from selected images with configurable slide duration and optional background music. The video is generated by FFmpeg and saved as a cataloged asset.

### Modified Capabilities

_(none)_
